Detailed Description

The CLGLTask class is responsible for shared tasks. The shared tasks are the tasks that utilize OpenCL-GL interoperability.

unsigned int  atomsnumber,
float  interactiondistance,

unsigned int  localsize = 0 
)

ComputeProteinLipidInteraction this function utilize the shared context to perform the protein-lipid interaction test.

Parameters
vertexbufferidthe id of a valid OpenGL vertext buffer
colorbufferidthe id of a valid OpenGL color buffer
systemstructurea void pointer to SystemStructureRecord structure. This structure is used in the GPU.
proteinspacesa void pointer to ProteinSpace structure. This structure is used in the GPU.
proteinparticlesa void pointer to ProteinParticle structure. This structure is used in the GPU.
atomsnumbernumber of atoms involved in the interaction test
interactiondistancea float point number to specify the maximum distance of the interaction
workitemssizespecifying the number of work items to be excuteed in the GPU. Usually we use the atoms number.
localsizespecify the number of items to be excuted within each group in the GPU.
